### Bézier Curves
A Bézier curve is a parametric curve and has a set of discrete "control points" which define a smooth and continuous curve. This is done by means of a formula. There are several orders of Bézier curves (Linear, Quadratic, Cubic etc.). For formulas and more info see this wiki article ([Bézier Curves wiki](https://en.wikipedia.org/wiki/B%C3%A9zier_curve)).
